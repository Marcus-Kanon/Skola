Solve the exercises using media queries. You may not modify the HTML.
Tip: use the CodePen view that shows the preview to the left/right of the code windows. You can easily check your queries by dragging the center bar.
Use :nth-child(n) or add a CSS class to select individual items, if you have to.
  1. Initially, display these instructions and all the fruits. Display the fruits in a column.
  2. Set a generous height to the column. Let the fruits expand to fill the available space. But the penguin requires double the height. (flexbox)
  3. Display the fruits in 3 columns (grid)
  4. ... and move the penguin to the middle. (Between pear and melon in the grid.)
  5. Use flexbox if the viewport is less than 500 pixels, grid otherwise.
  6. Display the "liten" element if the width is less than 400 pixels.
  7. Display the "stor" element if the width is greater than 600 pixels. Keep the previous CSS.
  8. Display the "mellan" element if the width is between 300 and 700 pixels. Keep the previous CSS.
  9. Modify the CSS so that at most one of liten/mellan/stor is visible at any time.
  10. Refine the grid so that it uses 2, 3 or 4 columns; depending on the available width.
  11. Make a menu bar with the options: Home, Shop, About. The menu items should display in a row if there is enough space, in a column otherwise.
